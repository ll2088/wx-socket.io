### 非常简单的socket.io聊天室   同时支持微信小程序终端，和浏览器客户端。

#### 1.克隆代码到本地后，进入WebSocketServer目录执行node index.js。

#### 2.使用 “微信开发者工具” 将 weapp_demo 目录打开，运行调试即可。

#### 3.浏览器可以访问WebSocketServer/chat目录中index.html文件。
